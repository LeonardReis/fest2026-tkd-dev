## 🚀 Como Executar Localmente

**Pré-requisitos:** Node.js (v18+)

1. **Instalar Dependências:**
   ```bash
   npm install
   ```
2. **Configuração de Ambiente:**
   Configure as chaves e credenciais no arquivo `.env`.
3. **Iniciar Ambiente de Desenvolvimento:**
   ```bash
   npm run dev
   ```
